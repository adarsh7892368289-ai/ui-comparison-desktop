# UI Comparison Desktop App

Desktop application for automated UI screenshot comparison, style extraction, and report generation.

## Features
- Visual diff comparison with severity analysis
- CSS attribute extraction & normalization 
- Selector generation (CSS/XPath)
- Multi-format export (JSON, CSV, HTML, Excel)
- Playwright-powered browser automation
- IndexedDB persistence

## Tech
- Electron + Webpack
- Vanilla JS (no frameworks)

## Setup
```bash
npm install
npm start
```
